# Contributing to AURA Testnets

## Scope

This repository contains network configurations for AURA testnets. For core development, see the [main AURA repository](https://github.com/aura-blockchain/aura).

## How to Contribute

- **Genesis Transactions**: Submit gentx during genesis ceremony
- **Peer Lists**: Add your node to peers.txt via PR
- **Documentation**: Improve joining guides and sync instructions

## Code of Conduct

See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).

## Questions
